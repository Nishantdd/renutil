## Build From Source

Prerequisites:
- [Node.js (LTS recommended)](https://nodejs.org/en/download)
- [Rust](https://rust-lang.org/tools/install/)

<br>

```bash
git clone https://github.com/Nishantdd/renutil.git
cd renutil
pnpm install
pnpm tauri build
```

## Development
